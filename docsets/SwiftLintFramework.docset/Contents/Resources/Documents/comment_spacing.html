Comment Spacing

Prefer at least one space after slashes for comments

  • Identifier: comment_spacing
  • Enabled by default: Yes
  • Supports autocorrection: Yes
  • Kind: lint
  • Analyzer rule: No
  • Minimum Swift compiler version: 5.0.0
  • Default configuration: warning
